<HTML Tags>

How to use HTML tags?

All HTMl Tags are surrounded by two characters < and >

The characters that surround HTML tags are called Angle Brackets

You can type Angle Brackets by pressing Shift + < or >

HTML tags normally come in pairs like <b> and </b>

The first tag in a pair is the start tag, the second tag is the end tag

The End tag must contain a </>

The text between the start and end tags is the element content

HTML tags are not case sensitive, <b> means the same as <B>

HTML Document Structure

At the beginning of every HTML document using HTML5 must include <!DOCTYPE html>

Everything goes in between the <html> tags

The <head> tags are for the website metadata such as title and description

All of your website content goes in between the <body> tags

Try It

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is where your main code goes</h1>
    </body>
</html>

Header tags

Header tags are typically used as titles

The header tag starts with an h and ends with a 1-6 ex. <h2> <\h2>

<h1> is the largest text size and <h6> is the smallest text

All Header tags require a closing tag

Try It

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>

        <h1>Extra Large</h1>
        <h2>large</h2>
        <h3>Normal</h3>
        <h4>Small</h4>
        <h5>Smaller</h5>
        <h6>Smallest</h6>

    </body>
</html>

Paragraph Tag

The use of a paragraph tag is right in its name, it is to display blocks of text

The paragraph tag is <p> </p>

The paragraph tag text is at the small size or <h6> to reference headers

To make a new line use <br> at the spot you want to make a new line, it does not need a closing tag

The paragraph tag requires a closing tag

Try It

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>

        <p>This is a paragraph tag it also has no br tag causing it to create a new line once it reaches the limit of its container </p>

        <p>By using the br tag i can create a new line to keep format as i see in the IDE<br></p>

    </body>
</html>

Anchor Tags

The anchor Tag is used to embed links into your website

The anchor tag is <a herf=" "> </a>

The anchor tag has one in-line parameter herf=""

The herf parameter is used to specify the url to a website or an id for an internal link

Try It

<!DOCTYPE html>
<html>
    <head>
    <title>Page Title</title>
     /head>
    <body>
       <a href="index.html">This will take you to the home page</a>
       <a href="#paragraph">This will take you the paragraph below</a>
    </body>
</html>

HTML Lists

there are two different types of lists in html

the first list is <ul>for an unordered list

the first list is <ol>for an ordered list

the the ordered list is used for numbering such as 123 etc or roman numerals

the unordered list is used for dot jot squares etc or nothing

in order to put data inside your list you have to use the <li> </li> inside the list tags

Try It

<html>

    </head>
    <body>
        <ul>
            <li>first item
            <li>second item

Tables

The table tag is used to create tables of what ever size you want

the table has four tags required to use it

The first Table tag is <table></table> ths is ued to define that table body just like the <body> tag

the second table tag is <tr></tr> tag witch is used to make a table row

the third table tag is <td></td> tag witch is used to add your data/text to the table

As you add table data it will make a new colum per tag

the fourth table tag is <th></th> tag witch is ued to make the table header it is not necessary but it bold's the text

Comments

Comment tags are extremely useful for making your code more readable and easier to debug

Comment tags make no difference to your code

The comment tag syntax is <!-- Your Text Here -->

Normally dies will grey out comments to show they are not part of the normal code

Try It

    </head>
    <body>
